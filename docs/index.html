Common questions

Frequently asked

What is Fleece?
Fleece is a credit card research CLI and AI chatbot. It uses live Brave Search data to answer questions about fees, rewards, transfer partners, statement credits, and more — so you never rely on outdated model training data.
How do I install it?
pip install fleece-cli — then set your BRAVE_API_KEY environment variable and run fleece --help.
Does it work with AI agents?
Yes. Every command supports a --json flag that returns structured output. Skills are also published to ClawHub so Claude Code, Codex, and other agents can use Fleece directly without any extra wiring.
Which issuers are supported?
All major US issuers: Amex, Bank of America, Barclays, Bilt, Capital One, Chase, Citi, Discover, Robinhood, U.S. Bank, Wells Fargo, and more. If Brave can find it, Fleece can research it.
Is there a web or chat interface?
Yes — the Fleece Streamlit app provides a conversational interface powered by OpenAI models with entity memory. Clone the repo and run streamlit run fleece.py.
Is it open source?
Yes. Fleece is MIT licensed. Find it on GitHub.
